# Amazon clone web app

Live demo: https://clone-3edc9.web.app

![plot](./images/home.PNG)

A single page application built using reactjs to browse amazon like website with stripe payment integration.

**Functionality**:

- Browse item and add them to basket
- Google auth
- Stripe payment
- View Order history

**Build**:

clone the repo: `https://github.com/nareshNishad/Amazon-clone`

Install dependency: `npm install`

Start frontend: `npm start`
